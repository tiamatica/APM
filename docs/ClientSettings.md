# Client Settings

There is a configuration file that keeps track of global settings.

## Registries

```json
{
    "registries": [
        {
            "alias": "local",
            "uri": "file://C:/Users/me/myregistry",
        },
        {
            "alias": "dyalog",
            "uri": "https://packages.dyalog.com",
        },
        {
            "alias": "myCompany",
            "uri": "https://packages.my-company.com",
            "user": "me@my-company.com"
        }
    ]
}
```
